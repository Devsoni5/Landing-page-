Description

The Harrier is Tata's premium five-seat SUV. It is the first car from the company to be built using a Land Rover platform, and it has a front-wheel-drive layout and monocoque construction. Launched in 2019 in diesel-manual form only, the Harrier now has a diesel-auto version too.
